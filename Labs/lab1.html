Lab 1

The purpose of this lab is to setup the Ardunio IDE and become familiar with the Artemis board. Additionally, we are focused on establishing BLE connection, so we can send and receive messages via Bluetooth. We also want to test all the hardware on the Artemis such as the temperature sensor and microphone.

Prelab

The first step was to install all necessary software to support the Artemis. Also, it was necessary to acquire all the hardware to connect the Artmeis to our personal laptops.
